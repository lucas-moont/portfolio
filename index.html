Gym API

Node.js | Vitest | Fastify | PostgreSQL | Docker | Code

Solução de plataforma para academias que permite aos usuários realizar o checkout de planos de forma dinâmica, com a restrição de geolocalização.
Além disso, a plataforma possui um controle de permissões bem definido, onde apenas administradores têm acesso à funcionalidade de registrar, editar e excluir academias. Esse controle de acesso foi implementado para garantir que a administração da plataforma seja segura e restrita.
Para assegurar a qualidade do código e a estabilidade da API, foram realizados testes automatizados com Vitest.

Restaurante API

Node.js | Express | SQLite | Knex | Code

A Restaurant API é uma solução desenvolvida para gerenciar restaurantes de forma eficiente, permitindo o cadastro de mesas e produtos, além da atualização e exclusão de itens do menu. Com ela, é possível abrir e fechar contas vinculadas às mesas, garantindo um controle preciso dos pedidos. Construída com Node.js e Express, a API utiliza SQLite como banco de dados, sendo gerenciado via knex para maior praticidade.
